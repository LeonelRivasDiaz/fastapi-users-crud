#FastAPI Users CRUD

API REST desarrollada con **FastAPI** que implementa un CRUD completo de usuarios utilizando **SQLAlchemy** y **SQLite**.



##Tecnologías utilizadas

- Python
- FastAPI
- SQLAlchemy
- Pydantic
- SQLite
- Uvicorn

---

## Funcionalidades

- Crear usuarios
- Listar usuarios
- Obtener usuario por ID
- Actualizar usuarios
- Eliminar usuarios
- Validación de datos con Pydantic
- Documentación automática con Swagger

---
## Inicializar el proyecto
- uvicorn main:app --reload
